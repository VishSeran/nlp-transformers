## 📌 About This Repository

This repository is designed as a **step-by-step learning path** to master NLP by **building everything from scratch** — starting from basic preprocessing to advanced Transformer models like BERT.

💡 Perfect for:

* Students 👨‍🎓
* AI Enthusiasts 🤖
* Developers 👨‍💻

---

## 🧠 Learning Roadmap

```bash
Text Processing → Embeddings → Language Models → Transformers → LLM Applications
```

---

## 📂 Project Structure

```
nlp-transformers/
│
├── 01_text_preprocessing/
├── 02_embeddings/
├── 03_language_models/
├── 04_transformers/
├── 05_llm_applications/
│
├── utils/
└── requirements.txt
```

---

## 🔍 Modules Breakdown

### 🟢 01. Text Preprocessing

* Tokenization
* One-hot Encoding
* Bag of Words

### 🔵 02. Embeddings

* Word2Vec (from scratch)
* PyTorch Embeddings
* EmbeddingBag

### 🟣 03. Language Models

* N-gram models
* RNN-based models

### 🔴 04. Transformers

* Attention Mechanism
* Transformer (from scratch)
* BERT Fine-tuning

### 🟡 05. LLM Applications

* Text Classification
* Named Entity Recognition
* Text Generation
